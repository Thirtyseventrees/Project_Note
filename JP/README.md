# 动词分类

## 一类动词（五段动词）

1. 结尾是う段且不是る的动词
2. あ段+る、う段+る、お段+る
3. 特例：切るーきる、帰るーかえる、走るーはしる、知る-しる  
   通常单个汉字+る的且汉字读音在い段或え段上的动词都是五段动词
    | 动词 | 读音   | 
    | -- | ---- |
    | 切る | きる   | 
    | 走る | はしる  |
    | 入る | はいる  |
    | 知る | しる   | 
    | 要る | いる   | 
    | 帰る | かえる  |
    | 減る | へる   | 
    | 喋る | しゃべる 
    | 蹴る | ける   | 


## 二类动词（上一段动词， 下一段动词）

い段+る、え段+る

## 三类动词（カ变，サ变）

くる、する  
名词 + する

# 动词变形

## ます形

### 一类动词

う段变い + ます  

例如：
- 書く　ー　かく　ー＞　書きます
- 吸う　ー　すう　ー＞　吸います
- 切る　ー　きる　ー＞　切ります

### 二类动词

去掉る + ます

例如：
- 教える　ー　おしえる　ー＞　教えます
- 開ける　ー　あける　ー＞　開けます

### 三类动词

くる　ー＞　きます  
する　ー＞　します

---

### 肯定否定现在过去形式

现在或者未来　ます  
- 森さんは毎日はたらきます

现在或者未来否定　ません  
- 田中さんは今日働きません

过去时　ました  
- 森さんは先週休みました

过去否定　ませんでした  
- 私は昨日働きませんでした

## て形

### 一类动词

す　ー＞　して
- 話す　ー　はなす　ー＞　話して

く　ー＞　いて、　ぐ　ー＞　いで
- 磨く　ー　みがく　ー＞　磨いて
- 急ぐ　ー　いそぐ　ー＞　急いで

う、つ、る　ー＞　って
- 洗う　ー　あらう　ー＞　洗って
- 待つ　ー　まつ　ー＞　待って
- やる　ー＞　やって

ぶ、む、ぬ　ー＞　んで
- 遊ぶ　ー　あそぶ　ー＞　遊んで
- 読む　ー　よむ　ー＞　読んで
- 死ぬ　ー　しぬ　ー＞　死んで

特例:  
行く　ー　いう　ー＞　行って

### 二类动词

将る换成て
- 食べる　ー　たべる　ー＞　食べて

### 三类动词

する　ー＞　して  
くる　ー＞　きて

### 动词て型 + 动词 表示动作相继发生

两个或两个以上的动作按照时间顺序相继发生时使用动词て型

- 昨日でぱーどへ行って、買い物しました
- 家へ帰って、宿題をします

### 动词て型 + ください（下さい） 表示请求某人做某事

- ここに住所と名前を書いてください

### 动词て型 + も + いいです　表示许可

- 家で仕事をしてもいいです
- ここで写真を撮ってもいいですか？

### 动词て型 + は + いけません　表示禁止

- 教室で物を食べてはいけません

### 动词て型 + いる　表示动作正在进行或者动作结果的持续状态

1. 表示动作正在进行
     - 彼は今、本を読んでいる
     - 森さんは今仕事をしています

2. 表示动作结果的持续状态  

   用于瞬间性变化动词  
   发生->结束->结果持续  
   - 森さんは車を持っています
   - 田中さんは知っていますか？

口语中经常会省略いる而直接在て后面加上る
- 読んでいる -> 読んでる
- 何している -> 何してる

## た形

变形方式和て型一致，只是把て换成た

## ない形

ない形可以理解为动词的否定形式  
本质上是动词未然形 + ない

### 一类动词

词尾的う段变成あ段 + ない  
但是う特例：うー＞わ

- 行く　ー＞　行かない
- 書く　ー＞　書かない
- 話す　ー＞　話さない
- 買う　ー＞　買わない

### 二类动词

去掉る + ない

- 食べる　ー＞　食べない
- 見る　ー＞　見ない
- 教える　ー＞　教えない
- 起きる　ー＞　起きない

### 三类动词

する　ー＞　しない
来る（くる）　ー＞　来ない（こない）

### 动词ない形 + で　+ ください 表示请不要做某事

表示否定的命令时使用～てください的否定形式：动词ない形+で+ください

- 部屋の鍵を忘れないでください
- この部屋には入らないでください
  - 这句话中用には表示强调前面的“この部屋に”，把“这个房间”拿出来当话题/强调/对比。实际上只用に也是可以的

# 动词的名词形式

**“动词基本型 + こと”可以作为名词使用：**

- 私の趣味は切手を集めることです　(趣味ーしゅみ、　切手ーきってー邮票、　集めるーあつめるー收集，聚集，招集)
- スミスさんはピアノを弾くことができます　(弾くーひく)
  - できます既可以表示做这个动作的能力，也可以表示做这个动作的许可，可不可以

过去的动作做名词用た型  
例如：动词た型 + ことが　+ あります　表示曾经~过
- 私わすき焼きをたべたことがあります

# 助词に

## 到达点/附着点

动作落到哪里，指向哪里  
动作是附着在这个物体上的（例如坐在椅子上）

典型句型：
- （Ｎ）に　行く/来る/帰る：去/来/回到某处
  - 学校（がっこう）に行く
  - 家に帰る
- （Ｎ）に　入る（はいる）/乗る（のる）/座る（すわる）/置く（おく）：进入/乘上/坐到/放到
  - 部屋に入る
  - 電車に乗る
  - 椅子に座る
  - 机（つくえ）の上に置く

に和へ对比：
- に更像是确实到达/落点明确
- へ更像是朝着...方向（方向感更强，不强调到没到）
  - 日本へ行く。（去日本/朝着日本去）
  - 日本に行く。（去日本，落点更明确）

に和を对比
- に一般用于目标点
  - 部屋に入る
- 而を用于“经过/离开”的场所
  - 道を歩く
  - 家を出る

## 存在点（人/物在哪里）

某个东西/人存在于这个点

典型句型：
- （Ｎ）に　ある/いる：在某处有（物用ある，人/动物用いる）
  - 机の上に本がある
  - 教室（きょうしつ）に先生がいる

常见扩展动词（也带着“停留/存在”的感觉）：
- 住む（すむ）：居住在
  - 中国に住んでいる
- 勤める（つとめる）：就职于
  - 会社に勤めている
- 残る（のこる）：留在
  - 家に残る
- 泊まる（とまる）：住/留宿在
  - ホテルに泊まる

## 归属点

某个状态，变化，关系归到哪里/落在谁身上

- 动作的对象（对谁做某事/谁是接受者）
  - 先生に聞く      问老师
  - 友達に会う      见朋友
  - 母に電話する    给妈妈打电话
  - 彼に謝る        向他道歉

- 授受（给/从）
  - 友達に本をあげる 把书给朋友
  - 友達に本をもらう 从朋友那里得到书
  - 先生に教えてもらう 请老师教我  

---

注意：助词を用于动作的对象，和に的区别是：
  - を：被直接作用/处理/改变的东西（直接受影响）
  - に：动作指向的点/对象/对方  

例如：

を：直接宾语（你对他做了事，他被影响）
- ご飯を食べる
- ドアを開ける  

处理X/改变X/把X怎样了

に：间接对象，动作的目标/对手/指向对象
- 友たちに本を上げる
- 母に電話する

不是被处理物，而是动作落在谁身上

---

- 变化结果（变成）
  - 医者になる 成为医生
  - 水が氷（こおり）になる  水变成冰
  - 部屋をきれいにする 把房间弄干净

- 决定的落点（定在.../选在）
  - 来週にする  定在下周
  - 3時に会おう 三点见
  - これに決める（きめる）  就决定用这个

## 表示动作发生的时间

名词 + に +　动词

- 森さんは七時に起きます

**注意：** 叙述包含数字的时间后面都要加に，但是“今、昨日、今日、明日、毎日、去年、来年”等词后面不能加に，星期后面可加可不加

# 助词で

## 场所：动作发生的地点

- 図書館（としょかん）で勉強（べんきょう）する
- 家でご飯を食べる
- 公園（こうえん）で遊ぶ

これは明日会議で使う資料です  
这句话中的で表示的是使用资料的场所是明天的会议，所以用で

## 手段：用...（工具/方法/交通工具/语言）

- 箸で食べる
- 電車で行く
- 日本語で話す
- クレジットカードで払う（はらう）

## 材料：用...做成（原料/材料）

- 木（き）で机を作る（つくる）

## 原因，理由：因为...

- 雨で試合（しあい）が中止（ちゅうし）になった
- 病気（びょうき）で学校（がっこう）を休む

## 范围：在...范围内/以...为界（数量，时间，条件）
- 三人で行く
- 1日で終わる
- １００円で買える

## 状态：以...状态（保持某种样子做事）

- 裸足（はだし）で歩く（あるく）
- このままでいい
- 元気でよかった

# 助词を

## 直接宾语：把~当作被处理/被改变的对象

如果“X 会因为这个动作而被消耗/改变/处理”，大概率用 を。

- ご飯を食べる
- 水を飲む
- 本を読む
- ドアを開ける
- 日本語を勉強する

## 表示“离开点/经过点/路径”

- 家を出る
- 電車を降りる
- 道を歩く
- 橋を渡る（わたる）
